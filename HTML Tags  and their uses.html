< Tags > What they are used for
< !DOCTYPE Tag > They are used for defining the document type
< Article Tag > They are used in defining an article
< Address Tag > They are used to define the contact information of the author or owner of a documentd
< Base Tag > Specifies the base URL/target for all URLs in a document
< BDI Tag > Its isolates a part of text that might be formatted in a different direction from other text outside it
< BDO Tag > They are used to override the current text direction
< Blockquote Tag > They are used to define a section that is quoted from another source
< Body Tag > They are used to define the document's body
< BR Tag > They are used in making a single line break
< Button Tag > They are used in making clickable buttons
< Canvas Tag > They used to draw graphics, on the flier, via scripting ( usually JavaScript )
< Caption Tag > They are used in making a table caption
< Center Tag > They are used to make words or text at the center
< Col Tag > They are used in specializing column properties for each column within a < colgroup > element
< colgroup > They are used in specifing a group of one or more coloumns in a table for formatting
< Data > They are used to add a machine readable translation of a given content
< Datalist > They are used to sprcify a list of pre defined options for input controls
< Del > They are used to define a text has been deleted from a document
< Details > They are used to define additional details that the user can view or hide
< Div > They are used to define a section in a document
< Dl > They are used to define a description list
< Em > They are used to emphasize text
< Embed > They are used to define a container for an external application
< figure > They are used to specify a self contained content
< Footer > Define a footer for a document or section
< Form > Defines an HTML form for user input
< h1 to h6 > Defines HTML headings
< head > Contains metadata or information for the document
< Header > Define a header for a document or section
< HTML > Defines the root of an HTML document
< I > Define the root of HTML document
< IMG > Defines an image
< Iframe > Define an inline frame
< Input > Define an input control
< INS > Defines a text that has been inserted into a document
< KBD > Define a keyboard input
< Label > Defines a label for an < input > element
< Legend > Defines a caption for a < fieldset >
< li > Defines a list item
< Link > Define the relationship between a document and an external resource ( most used to link the stylesheet )
< Main > Specifies the main content of a document
< Map > Defines an image map
< Mark > Defines marked or highlighted text
< Meta > Defines metadata about HTML document
< noscript > Defines an alternate content of users that do not support client side scripts
< option > Defines an option in a drop down list
< output > Defines the result of a calculation
< p > Defines a paragraph
< Piture > Defines a container for multiple image resources
< Progress > Represents the progress of a task
< q > Define a short quotation
< s > Defines text that is no longer correct
< select > Defines a section in a document
